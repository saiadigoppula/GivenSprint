Semicolon in scripts

In this tutorial section you will learn about Semicolon in servicenow script

Where Semicolon is optional,Recommend and avoided

Required

Semicolon is strongly required in this syntax

var x = 0; x++;

gs.log(x)

OUTPUT

1

Optional

Semicolon is Optional in this syntax

var i;                        // variable declaration
i = 10;                        // value assignment
var x = 10;                    // declaration & assignment
var fun = function() {...};   // var decl., assignmt, and func. defin.
alert("hello");                  // function call

OUTPUT

Avoid and NO semicolons after

Semicolon is Optional in this syntax

if (...) {...} else {...}
for (...) {...}
while (...) {...}

OUTPUT

do while loop Semicolon

Unlike if,for,while conditions and loop statements do{...} while() loop need Semicolon

do {...} while (...);
